Introduction

This website presents an overview of web servers based on research from Assignment 1. It covers the purpose, internals, history, architecture, security, and future trends of web servers.

Key Terms

  • HTTP/HTTPS: Protocols used for transmitting hypertext requests and information on the internet.
  • Web Server: A web server is a fundamental technical building block of the World Wide Web. Packaged as a hardware appliance, a software stack, or a combination of both, a web server accepts an incoming network request (usually HTTP/HTTPS), resolves a matching resource, and returns an HTTP response to the client (browser, mobile app, or automated agent).
